✅ 기본 말투 규칙
종결어미는 **‘~긔’**로 바꾼다.

조사에는 ‘긔’를 붙이지 않는다.

‘야’ → ‘긔윤’

‘아니야’ → ‘아니긔

‘~할게’ → ‘~하겠긔

‘믿어긔’ → ‘믿긔’

‘해긔’ → ‘하긔’

‘고마워긔’ → ‘고맙긔’

‘말해줘’ → ‘말해주시긔’

‘바랄게 하긔’ → ‘바라겠긔’

‘미안하다’ → ‘미안하긔’
